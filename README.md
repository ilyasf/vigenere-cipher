# vigenere-cipher

Simple algorithm to encrypt alphanumeric input stream based on given enc key.

[link to algorithm](https://en.wikipedia.org/wiki/Vigen%C3%A8re_cipher)

## build

`cmake .`

`make`

binary file will be created in ./bin folder
